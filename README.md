# Java Florist Frontend

A responsive and professional frontend website for Java Florist, built with HTML, CSS, and JavaScript using Bootstrap.

## Features

- Home page with occasion links
- About, Contact, Occasions pages
- Bouquet listing with add to cart functionality
- Cart management
- Checkout with delivery details and payment options
- User account pages (Profile, Orders, Return Requests)
- Florist registration
- User login and registration

## File Structure

- `index.html` - Home page
- `about.html` - About Flowers
- `contact.html` - Contact information
- `occasions.html` - List of occasions
- `bouquet.html` - Bouquet listings
- `cart.html` - Shopping cart
- `checkout.html` - Checkout process
- `my-account.html` - Account dashboard
- `my-profile.html` - User profile
- `my-orders.html` - Order history
- `my-return-requests.html` - Return requests
- `florist-registration.html` - Florist registration
- `login.html` - User login
- `register.html` - User registration
- `css/style.css` - Custom styles
- `js/script.js` - General scripts
- `js/cart.js` - Cart functionality
- `js/checkout.js` - Checkout scripts
- `images/` - Placeholder for images

## Setup

1. Open `index.html` in a web browser to view the site.
2. Add actual images to the `images/` folder (e.g., birthday.jpg, wedding.jpg, etc.).
3. For dynamic functionality, integrate with backend later.

## Notes

- Cart data is stored in localStorage.
- This is a static frontend; backend integration needed for full functionality.
- Responsive design using Bootstrap.
